### "Website won't load"
→ Check: File paths in `index.html` match your folder structure

→ Check: CSS files in the `CSS` folder are properly linked

### "JavaScript errors"
→ Check: JS files in the `JS` folder are properly linked

### "Content doesn't update"
→ Check: Refresh browser (F5) after editing files

→ Check: Settings → Pages has correct branch and folder selected
