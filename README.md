## 📝 Customization

### Edit Colors
Modify CSS variables in `CSS/main.css`:
```css
:root {
	--gold-primary: #FFD700;
	--red-primary: #DC143C;
	/* ... more variables */
}
```

### Add Products
Edit `products` array in `JS/main2.js`:
```javascript
const products = [
	{
		id: 1,
		title: 'Your Product',
		description: 'Description here',
		price: '$99.99'
	}
	// ...
];
```

### Add Blog Posts
Edit `blogPosts` array in `JS/main2.js`:
```javascript
const blogPosts = [
	{
		id: 1,
		title: 'Your Article',
		category: 'Category',
		date: 'Date',
		readTime: '5 min read',
		content: 'Article content...'
	}
	// ...
];
```
