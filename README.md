# VitalTech Hub

A one-page affiliate marketing and blog website built with Tailwind CSS, Lucide icons, and plain JavaScript.

## Files

- `index.html` — main site page
- `styles/styles.css` — custom styling and gold accent theme
- `script/script.js` — interactive filters, modal, and newsletter subscription behavior
- `assets/` — images, icons, and fonts for site content
- `404.html` — GitHub Pages fallback page
- `robots.txt` — crawler instructions
- `sitemap.xml` — search engine sitemap

## Notes

- The site is designed for static hosting and works without a build step.
- Ensure `styles/styles.css` and `script/script.js` remain in their current folders.
