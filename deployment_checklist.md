# Deployment Checklist 🚀

Fulfill these steps to host your website on GitHub Pages.

- [ ] [cite_start]Check that `index.html` properly references `css/style.css` and `js/script.js`.
- [ ] Initialize a local Git repository: `git init`.
- [ ] Stage all workspace components: `git add .`.
- [ ] Commit tracking changes: `git commit -m "feat: gold pouch structural launch"`.
- [ ] Create a new repository on GitHub.
- [ ] Link local assets to the cloud remote: `git remote add origin <URL>`.
- [ ] Push tracking configurations: `git push -u origin main`.
- [ ] Head to **Settings > Pages** on your GitHub repository.
- [ ] Select the `main` branch as your build source and hit **Save**.
