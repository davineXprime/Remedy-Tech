# Quick Reference Guide 📊

## Design Specifications
* **Primary Gold Hex**: `#d4af37`
* **Vibrant Glow Glow**: `#ffdf00`
* **Primary Layout Base**: White (`#ffffff`)
* **Grid Sizing**: 40px matrix pattern with `rgba(255, 68, 68, 0.07)` transparency limits.

## Key CSS Classes
* [cite_start]`.collage-item`: Custom structural 1x3 items running alternating `rotateLeftToRight` cycles.
* [cite_start]`.popup-overlay`: Handles modal rendering using an open scroll pass configuration (`overflow-y: auto`).
