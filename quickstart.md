# Quickstart Guide ⚡

Follow these instructions to configure and run the Gold Pouch platform locally in minutes.

## Local Execution
1. Clone the project files locally into your desired workspace directory.
2. Ensure your directory layout aligns with the core architecture (`/css`, `/js`, `index.html`).
3. Open `index.html` within your web browser to view the functional layout instantly.

## Verification Tasks
* [cite_start]Verify that scrolling works continuously while a popup window remains active.
* Move your mouse across the background grid to check the responsive neon coordinate shifting.
